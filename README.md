## 📌 Project Description
This project is a **Quiz Master Pro - Interactive Quiz Game** built as Task 03 for SkillCraft Technology Internship. 
It features multiple question types, real-time scoring, progress tracking, and detailed results with a modern UI.
## 🎯 Objective
To develop an engaging and responsive quiz application that tests user knowledge with instant feedback and a clean user experience.
## ✨ Features Implemented
- **Multiple Question Types**: Single Choice, Multiple Select, Fill in the Blanks
- **Progress Dots & Progress Bar**: Visual tracking of quiz progress
- **Real-time Score & Timer**: Track score and total time taken
- **Instant Feedback**: Correct answers turn green, wrong answers turn red
- **Skip Question Option**: Skip and move to next question
- **Detailed Result Screen**: Shows final score, correct/wrong count, and time taken
- **Performance Feedback**: Personalized message based on score percentage
- **Dark/Light Theme Toggle**: Switch themes for better readability
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Restart Quiz**: Play again with one click
## 🛠️ Technologies Used
- **HTML5** - Semantic structure and accessibility
- **CSS3** - Styling, Flexbox, Grid, Gradients, Animations
- **JavaScript (ES6)** - Quiz logic, state management, DOM manipulation
